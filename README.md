# Stratec Challenge 2026 (Android)

## What is this?

This is an Android app that simulates rocket travel between planets.

It calculates:

* Travel time
* Distances
* Planet positions
* Best launch windows (advanced stages)

---

## How to run the project (Android Studio)

### 1. Open the project

* Open **Android Studio**
* Click **Open**
* Select the project folder

---

### 2. Let Gradle build

* Wait for **Gradle Sync** to finish
* If prompted, click **Trust Project** or **Sync Now**

---

### 3. Run the app

* Connect a phone **or** start an emulator
* Click the green **Run ▶️ button**

---

### Stage Features

#### Stage 1

* Escape velocities for each planet

#### Stage 2

* The time it would take a hypothetical rocket to reach escape velocity for each planet

#### Stage 3

* Simple straight-line travel

#### Stage 4

* Shows planet positions after X days

#### Stage 5

* Finds best launch window (planets don’t move during flight)

#### Stage 6

* Advanced simulation (planets move during flight)
